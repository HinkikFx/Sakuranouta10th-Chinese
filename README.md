# サクラノ詩 10th Anniversary Edition 简体中文翻译

《樱之诗 -在樱花之森上飞舞-》10周年纪念版 AI 简体中文翻译，仅供学习交流使用。

## 关于本翻译

本翻译使用ChatGPT4.1进行AI翻译，可能存在翻译不准确、语句不通顺等问题。如果你在游玩过程中发现明显的翻译错误，欢迎提交 Issue 或 Pull Request。

## 使用方法

将 `script` 文件夹放置到游戏目录下，覆盖原有文件即可。
